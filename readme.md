# README

## OpenAI RAG demo

## Overview  

This repository contains a demonstratory for performing and testing RAG with OpenAI.

## Prerequisites

- A valid OpenAI API key is required.

> [!IMPORTANT]  
> Do not store the key in plain text. Provide it as an environmental variable called OPENAI_API_KEY.

## Features  

- Document embedding into a local, in-memory vector store
- LLM prompting using the vector store for Retrieval Augmented Generation
